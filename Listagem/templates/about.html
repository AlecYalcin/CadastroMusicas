Website desenvolvido por Alec Can Yalçin e Paulo de Tarso

  Este sistema opera utilizando tecnologias web, como HTML, CSS e JavaScript para a camada de interface (contando com Bootstrap v5.0); uma linguagem de programação server-side, como o Python; e para gerenciar as requisições do usuário, armazenar e recuperar informações no banco de dados e lidar com a autenticação de usuários, o projeto utiliza o framework Django.

  Os integrantes do programa o fizeram enquanto cursavam o 4º Ano de Informática no IFRN - Campus Caicó , e se dispuseram a participar de reuniões breves e assíncronas, contudo diárias, para discutirem o que ainda havia sido requisitado pelo professor da disciplina - Ricardo Kleber - que ainda não tinha sido concluído ou trabalhado.
