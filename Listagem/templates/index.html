  Este projeto de cadastro de músicas tem como objetivo permitir que os usuários cadastrem suas músicas favoritas e as compartilhem com outros usuários. Ele inclui as seguintes funcionalidades:

  • Listagem: Os usuários podem visualizar todas as músicas cadastradas no sistema, ordenadas por título, artista ou data de cadastro.
  • Busca: Os usuários podem buscar por músicas específicas através de um campo de busca, filtrando por título, artista ou gênero.
  • Autenticação: O projeto inclui um sistema de autenticação para garantir que somente os usuários registrados possam cadastrar e visualizar as músicas. Isso inclui o login e o registro de novos usuários.
